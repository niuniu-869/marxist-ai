## "说人话"硬约束（M1-M7）

任何 LLM 输出违反一条即视为无效，整段重跑。

### M1 · 不准改观点

- **核心范畴不可弱化、不可替代、不可去政治化**：阶级、剥削、专政、革命、暴力、压迫、矛盾、斗争、消灭、推翻——出现即必须保留。
- 不准把"无产阶级专政"改为"工人阶级领导"；不准把"暴力革命"改为"激进变革"；不准把"消灭私有制"改为"调整财产关系"或"缩小贫富差距"；不准把"国家机器"改为"公共管理系统"；不准把"资产阶级"改为"富人阶层"；不准把"无产阶级"改为"打工人"；不准把"打碎国家机器"改为"重组国家机器"；不准把"机会主义"改为"灵活务实"。
- 不准把作者的论敌观点（杜林、伯恩施坦、考茨基、马赫、巴枯宁等）和作者本人观点混淆。

### M2 · 不准发明事实

- 不补充原文未提的事件、数据、引文。
- **严禁现代政治类比**（不准类比互联网平台、当代党派、当代国际关系、当代领导人、当代社会现象）。
- 历史细节模糊时，写"19 世纪欧洲工人运动"等概括，**不**编具体事件。
- 不补全人物事迹（除非 glossary 已有）。

### M3 · 论战对象必须显名

- 段落隐指某论敌时，必须在 `gist` 中点出（"本段批评考茨基'议会道路'主张"）。
- 引文需在 `notes` 中说明 `speaker`（作者本人 / 引马 / 引恩 / 引论敌）+ `stance`（赞同 / 批判 / 中性引用）。

### M4 · 时空指代不得抽象化

- "我们的时代""当前的形势""最近的事件"——首次出现必须在 note 中**还原为具体年代/事件**。
- "巴黎工人""德国同志""俄国革命派"等称谓首次出现给注释。
- 同一指代后续出现可省略注释（避免重复注水）。

### M5 · 比喻必须保留 + 解释

- 核心比喻原句必须**字面保留**，绝不改写或"意译没"了。命中以下清单时强制保留：
  - "宗教是人民的鸦片" / "鸦片"作宗教比喻
  - "幽灵在欧洲游荡"
  - "枷锁" / "他们失去的只是锁链"
  - "国家机器" / "打碎旧的国家机器"
  - "上层建筑 / 经济基础"
  - "桥梁""道路""灯塔""火车头"等历史哲学比喻
- 比喻在 `notes` 中加 `metaphor` 类型注释，说明"原意是什么、为什么这么比"。

### M6 · 老译晦涩词给现代等价

- 老译里"鄙陋""庸碌""乡愿""畔""桎梏""藉以""遂尔""寻常""若乃""其实"等晦涩词：原句**不替换**，只在 `notes` 中以 `concept_archaic_translation` 类型给现代等价。
- 长句、欧化句法可以在 `plain_rewrite` 中拆分重组，但术语保留。

### M7 · 客观中立

- 禁止评价词："伟大""光辉""深刻""科学地证明""英明""正确""精辟"。
- 禁止抒情、表态、夹带评论。
- `gist` / `historical_context` 只陈述事实，不褒贬。
