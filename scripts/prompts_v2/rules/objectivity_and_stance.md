### 现代政治类比禁止

不准把 19/20 世纪事件、人物、组织与下列任何现代实体并列、类比、暗示：
- 互联网 / 数字平台 / 算法 / 大数据 / 人工智能
- 当代任何政党、政府、领导人
- 当代任何国际组织、国际事件
- 当代企业、企业家
- 当代社会议题（性别 / 种族 / 移民 现代框架）

如需说明读者熟悉的当代现象作类比，**禁止**。坚持用原文时代的语境。

### 不发明事实

- 不补全原文未提的事件、数据、引文、人物事迹
- 模糊处用"据现有材料"、"具体不详"、留空字段
- 不靠常识填空。常识也错误率高。

### speaker / stance 严格区分（防止把论敌观点当作者立场）

| speaker | 含义 | stance 配对 |
|---|---|---|
| author | 作者本人陈述 | self |
| quoted_marx | 引用马克思原话 | endorsed / criticized / neutral_citation |
| quoted_engels | 引用恩格斯原话 | 同上 |
| quoted_lenin | 引用列宁原话 | 同上 |
| quoted_opponent | 引用论敌原话（含间接转述） | criticized（绝大多数） |
| editor | 编者注 / 译者注 | neutral_citation |
| unknown | 出处不明的引用 | neutral_citation |

**核心硬约束**：作者引用论敌的话时，**句子级 speaker 必须是 quoted_opponent**。前端展示要让读者看到 `[引论敌]` 标签，**否则读者会把论敌的话当作者主张**。这是 M3 防翻车的关键。
